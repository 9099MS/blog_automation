📝 티스토리 자동 포스팅 프로그램 사용법

안녕하세요! 이 프로그램을 사용해주셔서 감사합니다.

1. 실행 전 준비사항 (최초 1회)

프로그램을 처음 실행하기 전에, 아래의 준비를 꼭 마쳐주세요.

  • Chrome 브라우저 설치: 이 프로그램은 Chrome 브라우저를 이용해 작동합니다. 컴퓨터에 Chrome이 설치되어 있는지 확인해주세요.
  • .env 파일 수정: 함께 제공된 .env 파일을 메모장으로 열어, "" 안에 자신의 정보를 정확하게 입력하고 저장해주세요.
    • TISTORY_ID: 티스토리 로그인 아이디 (이메일)
    • TISTORY_PW: 티스토리 로그인 비밀번호
    • TISTORY_BLOG_URL: "sporg.tistory.com"과 같은 블로그 주소
    • GOOGLE_API_KEY, OPENAI_API_KEY 등: 사용하려는 AI의 API 키
  • prompt.md 파일 수정 (선택 사항): AI의 글쓰기 스타일을 변경하고 싶다면, prompt.md 파일을 열어 지시문을 자유롭게 수정하세요.

2. 프로그램 실행

  1. app.exe 파일을 더블클릭하여 실행하세요. (잠시 후 웹 브라우저가 자동으로 열립니다.)
  2. 만약 브라우저가 자동으로 열리지 않으면, 직접 웹 브라우저를 열고 주소창에 http://127.0.0.1:5001 을 입력하여 접속하세요.
  3. 화면에 나타난 옵션을 원하는 대로 선택하고 '포스팅 시작하기' 또는 '스케줄 등록하기' 버튼을 누르면 자동 포스팅이 시작됩니다.

3. 프로그램 종료

프로그램은 백그라운드에서 계속 실행되므로, 사용이 끝나면 직접 종료해주어야 합니다.

  1. 키보드에서 Ctrl + Shift + Esc 키를 함께 눌러 작업 관리자를 엽니다.
  2. '프로세스' 탭 (또는 '세부 정보' 탭) 목록에서 app.exe를 찾습니다.
  3. app.exe를 마우스 오른쪽 버튼으로 클릭한 후, '작업 끝내기' (또는 '프로세스 끝내기')를 선택하여 프로그램을 완전히 종료합니다.
